## What is RAP?

RAP (Valoración de Aprendizajes Previos) is a conversational AI system that evaluates the prior competencies of applicants to graduate programs at Universidad Icesi. Instead of generic admissions processes, RAP conducts an adaptive interview-style assessment and produces a personalized report that maps each applicant's strengths and gaps to the specific competencies of their chosen master's program.

The system uses an AI agent powered by LangGraph and OpenAI to guide applicants through a structured, natural conversation — asking SABER PRO-style questions, evaluating responses in real time, and ultimately generating a detailed learning path recommendation.

## How it works

<Steps>
  <Step title="Select a program">
    The applicant chooses the master's program they are applying to (e.g., Maestría en Educación con Énfasis en TIC).
  </Step>
  <Step title="Share your background">
    The system collects professional background, years of experience, current role, and relevant skills through a conversational form.
  </Step>
  <Step title="Complete the assessment">
    The AI agent conducts an adaptive competency assessment using questions drawn from the program's question bank. Questions span multiple difficulty levels (1–10) and types: multiple choice, open-ended, scenario-based, and SABER PRO contexto.
  </Step>
  <Step title="Receive your report">
    A personalized report is generated with strengths, areas to improve, a tailored learning path, and recommended next steps — all aligned to the specific competencies of the program.
  </Step>
</Steps>

## Architecture overview

RAP is a full-stack application with a React frontend served alongside a FastAPI backend:

```
Frontend (React + TypeScript)
        │
        ▼
  FastAPI Backend
        │
   ┌────┴────────────────────────┐
   │                             │
VAPAgent (LangGraph)      PostgreSQL (Neon)
   │
   ├── KnowledgeBaseManager (ChromaDB)
   ├── QuestionBankManager (JSON)
   ├── EvaluationEngine
   └── ReportGenerator (OpenAI)
```

The frontend guides applicants through the assessment flow. All chat messages go to the `/api/chat` endpoint, which dispatches to the `VAPAgent` — a stateful LangGraph graph that manages conversation turns, selects questions, evaluates answers, and triggers report generation.
